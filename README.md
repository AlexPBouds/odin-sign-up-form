# Odin Sign-Up Form

## Purpose

This project was meant to show that I could work with a design file, and create something that looked like it while also being able to use form data. None of the information input is stored anywhere either (except possible in your cache).

## Functionality

All of the fields except for the phone are mandatory, and so the "Create Account" button will not accept the form unless the other inputs are filled. Once the information is filled out, pressing the "Create Account" button will show a pop-up displaying the information you typed in. This information is read from the form data using Javascript. Pressing the "Log In" button does not currently do anything, as this was not part of the scope of the project.
